# Specification

## Summary
**Goal:** Add a Careers section above the footer with job listings and integrate it into the navigation.

**Planned changes:**
- Create a Careers section component with "Join Our Team" headline and company culture introduction
- Display 5 job listings (Digital Marketing Specialist, Paid Ads Manager, Social Media Strategist, Web Designer/Developer, Sales & Client Success Manager)
- Each job card shows title, description, location (Remote/On-site), and an "Apply Now" button with golden yellow styling
- Implement fade-in animations using IntersectionObserver pattern
- Add "Careers" link to Navigation component that scrolls to the section
- Position Careers section above Footer in the page layout

**User-visible outcome:** Users can browse open positions in a new Careers section, see job details, and access the section via the navigation menu.
